Applicant Review System

A secure, role-based web app for UNF's Psychology Department to streamline applicant reviews, interviews, and decisions—replacing spreadsheets and long email threads.

  • Role: Team lead • Full-stack dev (ASP.NET Core, EF Core)
  • Stack: ASP.NET Core MVC · SQL Server · Azure App Service + SQL DB
  • Highlights: CSV/Excel import, auto reviewer/interviewer assignment, audit trail, email templates
GitHub Link

Overview

We met with stakeholders to map the existing workflow (spreadsheets + email), defined access roles, Data in screenshots is dummy data. then iterated toward a system that supports file uploads, automated assignments, and filtering by review/interview status.
